# Random Task Generator

## Автор
Алексей Кутовой

## Описание проекта
Random Task Generator - это графическое приложение на Python с использованием библиотеки Tkinter.

Программа позволяет:
- Генерировать случайные задачи
- Добавлять собственные задачи
- Фильтровать задачи по категориям
- Просматривать историю сгенерированных задач
- Сохранять историю в JSON-файл

## Используемые технологии
- Python
- Tkinter
- JSON
- random
- Git

## Функции программы
### Генерация случайной задачи
Пользователь может нажать кнопку "Сгенерировать задачу", после чего программа выбирает случайную задачу.

### Категории задач
Доступны категории:
- Учёба
- Спорт
- Работа

### Добавление новых задач
Можно добавить собственную задачу в выбранную категорию.

### Фильтрация
Можно фильтровать историю задач по категории.

### Сохранение данных
История автоматически сохраняется в файл book.json.

## Проверка ввода
Если пользователь пытается добавить пустую задачу программа выводит сообщение об ошибке.
